## Getting Started

### Prerequisites
Visual Studio Code or Visual Studio Version 2026

### Installation
Quick start commands:

1. Clone the repo
```bash
git clone https://github.com/OmegaFullard/PipeDreamsDev2.git
cd PipeDreamsDev2
```

2. Install dependencies (example for Node)
```bash
npm install
# or
yarn install
```

3. Start the app
```bash
npm start
# or
yarn start
```
